# Daily Wellness Tracker - Project Report

![Project Banner](wellness_tracker_banner_1776867304918.png)

## 1. Introduction
The **Daily Wellness Tracker** is a comprehensive full-stack web application designed to help users monitor and improve their physical and mental health. In an era where maintaining a healthy lifestyle is increasingly challenging, this tool provides a centralized platform for tracking essential wellness metrics, from physical activity to emotional well-being.

## 2. Purpose
The primary purpose of this project is to empower individuals to take control of their health through data-driven insights. By providing an intuitive interface to log daily habits, the application aims to:
- Encourage consistent hydration and physical activity.
- Provide a clear overview of nutritional habits.
- Foster mental health awareness through regular journaling.
- Simplify wellness management by aggregating multiple health data points into a single dashboard.

## 3. Techstack
The application is built using a modern MERN (MongoDB, Express, React, Node.js) architecture, ensuring scalability, performance, and a smooth user experience.

### Frontend
- **React**: Powering the dynamic and responsive user interface.
- **Tailwind CSS**: Providing a sleek, modern utility-first styling.
- **Framer Motion**: Implementing smooth micro-animations and transitions for a premium feel.
- **Recharts**: Delivering interactive data visualizations and health progress charts.
- **Lucide React**: Minimalist and consistent iconography.

### Backend
- **Node.js & Express**: Handling server-side logic and API routing.
- **MongoDB & Mongoose**: Providing a flexible and scalable NoSQL database solution.
- **JWT (JSON Web Tokens)**: Ensuring secure, stateless user authentication.
- **Bcryptjs**: Implementing industry-standard password hashing for user security.

## 4. Features
- **Comprehensive Dashboard**: A unified view of all wellness metrics, including water intake, steps, and calories.
- **Activity Tracking**: Dedicated modules for logging workouts and monitoring physical progress.
- **Nutrition Management**: Tools to track caloric intake and maintain dietary goals.
- **Hydration Tracker**: An interactive interface to log daily water consumption.
- **Task Management**: An integrated Todo list to organize health-related goals and daily habits.

## 5. Special Features
- **Fitbit Integration**: Seamlessly sync health data from Fitbit wearable devices to the tracker, reducing manual entry.
- **Edamam AI Food Analysis**: An advanced integration that allows users to type in any meal (e.g., "Avocado toast") and automatically fetch precise nutritional data including calories, macros, and fiber using the Edamam API.
- **Mental Wellness Journaling**: A dedicated space for users to reflect on your day, helping identify patterns in mood and energy levels.
- **Dynamic Analytics**: Advanced chart visualizations that transform raw health data into actionable insights over time.
- **Responsive Premium Design**: A mobile-friendly, "glassmorphism" inspired UI that remains functional and beautiful across all devices.
